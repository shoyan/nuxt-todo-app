# nuxt-todo-app

Nuxt.jsで実装したTODOアプリです。

![スクリーンショット](./screenshot.png)

# Required

* Node.js >= 8
* SQLite

## Build Setup

``` bash
# install dependencies
$ npm install

# データベースの作成
$ npx sequelize-cli db:migrate

# serve with hot reload at localhost:3000
$ npm run dev

```

Build Setupを実行後に `http://localhost:3000` にアクセスすると表示されます。
