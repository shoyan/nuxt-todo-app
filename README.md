# nuxt-todo-app

Nuxt.jsで実装したTODOアプリです。

![スクリーンショット](./screenshot.png)

## Build Setup

``` bash
# install dependencies
$ npm install

# データベースの作成
$ sqlite3 -init db/init.sql todo.db

# serve with hot reload at localhost:3000
$ npm run dev

```

Build Setupを実行後に `http://localhost:3000` にアクセスすると表示されます。
